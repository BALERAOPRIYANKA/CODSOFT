Overview of the Project

Project: Personal Portfolio Website using HTML & CSS

Objective
The objective of this project is to create a responsive personal portfolio website using HTML and CSS. The aim is to showcase my skills, education, contact details, and professional summary through a clean, modern UI design similar to professional portfolio templates.

Key Activities
- Structure Design: Creating proper HTML structure with Header, Hero, and About sections.
- Styling: Using CSS for layout, colors, fonts, and responsive design for mobile/desktop.
- Content Integration: Adding personal details, education info, and resume download option.
- Image Handling: Inserting and styling profile images for hero and about sections.

Technologies Used
- HTML5: The primary markup language for structuring the portfolio content.
- CSS3: Used for styling, layout, animations, and responsive design.
- VS Code: Code editor used for writing and testing the project.
- GitHub: Used for version control and hosting the project repository.

Key Insights
- Learned how to structure a multi-section webpage with semantic HTML.
- Understood CSS Flexbox for creating responsive layouts.
- Implemented sticky navigation and hover effects for better UX.
- Practiced mobile-first responsive design using media queries.

Output
A fully responsive portfolio website with:
1. Header Section: Logo and navigation menu
2. Hero Section: Name, tagline, intro, "Hire Me" button, and profile image
3. About Section: Detailed bio and "Download Resume" button

Conclusion
This project helped me strengthen my frontend development fundamentals using HTML and CSS. I gained hands-on experience in converting a design template into a working website and learned best practices for clean, maintainable code.
